  3. DISCLAIMER: This app and the information contained herein do not constitute any financial or investment advice.

Simple stock screener. Uses simple time-series forecasting. Top 100 trending stocks shown. The stocks are filtered from 8000+ stocks traded in the US.

Uses the following:

  • Finhubb financial API
  • Facebook Prophet library (Python)
  • ECharts.js
  • Python Flask on backend. Vanilla Javascript on the frontend.
  • Bootstrap 4 CSS Framework.
